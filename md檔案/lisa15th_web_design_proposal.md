# LiSA 15th 台北場粉絲網站設計提案

> 依據 `/素材` 內 5 張官方視覺與周邊圖整理  
> 建議定位：非官方粉絲資源站 / 演唱會預習入口 / 票袋下載與周邊資訊整合頁

---

## 1. 設計方向

### 核心概念

**「粉紅噪音拼貼 × 15 週年應援基地」**

網站不做一般演唱會資訊頁，而是做成一個帶有演唱會視覺語氣的粉絲入口。整體感覺要像使用者從社群貼文點進來後，立刻進入 LiSA 15th 的世界：亮粉色、黑白拼貼、草莓標誌、手寫塗鴉、粗糙紙張與舞台前的興奮感。

### 關鍵印象

- 高對比：黑、白、亮粉色為主，少量紅色線條點綴。
- 拼貼感：使用撕紙、網點、掃描質感、邊框線與手寫塗鴉。
- 應援感：倒數、歌單預習、注意事項、票袋下載、周邊資訊都放在同一條動線。
- 手機優先：粉絲多半從 IG、Threads、LINE 群組點入，第一版應以手機閱讀效率為主。

## 4. 視覺系統

### 色彩

| 用途 | 色彩 | 建議色碼 |
|---|---|---|
| 主色 | LiSA hot pink | `#ff2f9f` |
| 背景黑 | 深黑 | `#080708` |
| 紙張白 | 灰白 | `#f4f1ef` |
| 墨色 | 近黑 | `#171316` |
| 紅線點綴 | 手寫紅 | `#d8282f` |
| 銀灰 | 閃粉灰 | `#c8c8c8` |

### 字體

- 中文：`Noto Sans TC`
- 英文標題：`Bebas Neue` 或類似 condensed 字體
- 裝飾文字：使用圖片本身的手寫字，不另外仿寫，避免風格失真。

### UI 風格

- 按鈕：黑底白字 / 粉色描邊 / 小面積亮粉填色。
- 卡片：不做柔和大圓角，建議 4-8px 小圓角或直接直角。
- 分隔線：使用細線、虛線、撕紙邊、網點背景。
- 圖片：允許部分裁切、重疊與斜角，維持拼貼感。

## 6. 互動與動效

建議保持輕量，不做過度動畫。

- Hero 進場：背景圖微微放大，文字淡入。
- 倒數計時：數字每秒更新，但不做跳動特效。
- 歌單項目：點擊後開啟 YouTube，新分頁。
- Checklist：點選可打勾，讓使用者當天可直接使用。
- 周邊區：商品分類可用橫向滑動，手機友善。
- CTA：hover 時出現粉色描邊或底色反轉。

---

## 7. 內容語氣

整體文字要像粉絲整理給粉絲，而不是官方公告。

建議語氣：

- 清楚、短句、直接。
- 保留一點期待感，但不過度撒嬌。
- 重要資訊優先，裝飾文字少量即可。

範例：

- 「先把歌單熱好，進場就能一起喊。」
- 「粉紅色準備好，手機記得靜音。」
- 「票袋 A4 黑白列印即可，出門前 5 分鐘也能完成。」

## 10. 總結

這批素材非常適合做成「強視覺粉絲工具站」。最重要的不是新增很多裝飾，而是讓官方素材本身成為網站的主角，再用清楚的資訊架構承接粉絲真正需要的功能：倒數、歌單、注意事項、票袋、周邊。

建議下一步直接製作一版首頁 mockup 或 HTML 原型，先完成手機版第一屏與歌單區，確認視覺方向後再補齊整頁。
